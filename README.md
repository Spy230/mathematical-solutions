# Математические Примеры  

## Описание

**Математические Примеры** — это мобильное приложение, которое генерирует три случайных математических уравнения. Пользователь должен решить уравнения   и ввести свои ответы. При правильном решении задач и  нажатии кнопки "Обновить" приложение генерирует новые уравнения.

## Стек Технологий

- **Язык программирования**: Java
- **Платформа**: Android
- **Среда разработки**: Android Studio

## Функции

1. **Генерация Уравнений**:
   - Приложение отображает три случайных математических уравнения (сложение, вычитание, умножение).

2. **Решение Уравнений**:
   - Пользователь вводит ответы в текстовые поля.

3. **Обновление Уравнений**:
   - При нажатии кнопки "Обновить" приложение генерирует новые случайные уравнения и возвращает исходный цвет полей ввода.

# Математические Примеры 1.0

## Описание

**Математические Примеры 1.0** — это мобильное приложение, которое генерирует три случайных математических уравнения. Пользователь должен решить уравнения как можно быстрее и ввести свои ответы в соответствующие поля. При нажатии кнопки "Обновить" приложение генерирует новые уравнения и сбрасывает цвет полей ввода.

## Стек Технологий

- **Язык программирования**: Java
- **Платформа**: Android
- **Среда разработки**: Android Studio

## Функции

1. **Генерация Уравнений**:
   - Приложение отображает три случайных математических уравнения (сложение, вычитание, умножение).

2. **Решение Уравнений**:
   - Пользователь вводит ответы в текстовые поля.

3. **Обновление Уравнений**:
   - При нажатии кнопки "Обновить" приложение генерирует новые случайные уравнения и возвращает исходный цвет полей ввода.

4. **Обработка Кнопки**:
   - Используется `OnClickListener` для обработки нажатия на кнопку обновления.
